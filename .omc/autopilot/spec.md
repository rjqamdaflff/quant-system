# 股票量化交易系统 - 产品规格说明书

## 1. 项目概述

### 1.1 项目名称
Quant System - 股票量化交易系统

### 1.2 项目定位
基于Java Spring Boot + Python的A股量化分析平台，提供数据采集、策略研究、回测验证、信号预测和交易提醒功能。

### 1.3 核心原则
- **真实数据优先**：禁止模拟数据，网络不可用时抛出异常
- **模块化架构**：各模块独立，可单独部署
- **可扩展性**：支持自定义策略、自定义数据源

## 2. 系统架构

```
┌─────────────────────────────────────────────────────────────────┐
│                        前端界面 (Vue.js)                         │
│  ├─ 数据管理页  ├─ 策略研究页  ├─ 回测分析页  ├─ 信号提醒页      │
└─────────────────────────────────────────────────────────────────┘
                                │
                                ▼
┌─────────────────────────────────────────────────────────────────┐
│                        后端服务 (Spring Boot)                    │
├─────────────────────────────────────────────────────────────────┤
│ 模块1: 数据采集层    │ Python脚本调用AKShare/Baostock获取真实数据 │
│ 模块2: 数据存储层    │ MySQL存储，支持增量更新                    │
│ 模块3: 数据清洗层    │ 异常数据处理、缺失值填充、标准化            │
│ 模块4: 策略分析层    │ 技术指标计算、策略回测引擎                  │
│ 模块5: 预测提醒层    │ 信号预测、多渠道推送                        │
└─────────────────────────────────────────────────────────────────┘
                                │
                                ▼
┌─────────────────────────────────────────────────────────────────┐
│                        数据层                                    │
│  ├─ MySQL (股票数据、用户配置)                                   │
│  ├─ Python Scripts (数据采集)                                    │
│  └─ AKShare/Baostock (真实数据源)                               │
└─────────────────────────────────────────────────────────────────┘
```

## 3. 功能模块详细规格

### 模块1: 数据采集层 (已完成)

**状态**: ✅ 已实现

**功能**:
- 股票列表采集
- 日线数据采集
- 分钟线数据采集
- 财务数据采集
- 增量采集支持
- 重试机制
- 数据验证

### 模块2: 数据存储层

**功能**:
- 数据库表结构优化
- 数据索引优化
- 数据备份策略
- 数据清理策略（过期数据归档）

**数据表**:
- stock_info: 股票基本信息
- stock_daily: 日线数据
- stock_minute: 分钟线数据
- stock_financial: 财务数据
- strategy_config: 策略配置
- backtest_result: 回测结果
- signal_alert: 信号提醒

### 模块3: 数据清洗层

**功能**:
- 停牌数据处理（标记而非删除）
- 缺失值填充（前值填充、插值）
- 异常值处理（涨跌幅超限、价格异常）
- 数据标准化（复权处理）
- 数据完整性检查

### 模块4: 策略分析层

**技术指标**:
- 移动平均线: MA5, MA10, MA20, MA60
- MACD: DIF, DEA, MACD柱
- KDJ: K, D, J
- RSI: 相对强弱指标
- BOLL: 布林带
- 成交量指标: VOLUME, VOL_MA

**策略引擎**:
- 策略定义接口
- 策略回测引擎
- 收益评估（年化收益、最大回撤、夏普比率）
- 策略优化

**内置策略**:
- 均线策略（金叉买入、死叉卖出）
- MACD策略
- KDJ超买超卖策略
- 布林带策略

### 模块5: 预测提醒层

**信号预测**:
- 基于技术指标的信号生成
- 买卖点识别
- 风险预警（跌破均线、量价异常）

**提醒渠道**:
- 邮件通知
- 钉钉机器人
- 企业微信
- 飞书

**提醒规则**:
- 价格突破提醒
- 指标信号提醒
- 自定义条件提醒

## 5. API接口规格

### 5.1 数据采集API
- GET /api/collect/stocks - 获取股票列表
- POST /api/collect/daily/{code} - 采集日线数据
- POST /api/collect/daily/batch - 批量采集
- GET /api/collect/status - 系统状态

### 5.2 数据查询API
- GET /api/data/daily/{code} - 查询日线数据
- GET /api/data/stocks - 股票列表
- GET /api/data/financial/{code} - 财务数据

### 5.3 策略分析API
- POST /api/strategy/indicator - 计算技术指标
- POST /api/strategy/backtest - 执行回测
- GET /api/strategy/list - 策略列表
- POST /api/strategy/config - 配置策略

### 5.4 信号提醒API
- GET /api/signal/list - 信号列表
- POST /api/signal/rule - 创建提醒规则
- PUT /api/signal/rule/{id} - 更新规则
- DELETE /api/signal/rule/{id} - 删除规则
