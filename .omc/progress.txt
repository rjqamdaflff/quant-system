## API 接口

| 接口 | 方法 | 说明 |
|------|------|------|
| /api/collect/stocks | GET | 获取股票列表 |
| /api/collect/status | GET | 获取系统状态 |
| /api/collect/daily/{code} | GET | 查询日线数据 |
| /api/collect/daily/{code} | POST | 采集单只股票 |
| /api/collect/daily/batch | POST | 批量采集 |
| /api/collect/validate/{code} | GET | 验证数据 |

## 注意事项

1. **真实数据源**：系统通过Python脚本调用AKShare/Baostock获取真实数据
2. **禁止模拟数据**：网络不可用时系统会抛出异常，不会返回假数据
3. **数据验证**：所有入库数据都经过严格验证
4. **环境要求**：需要Python3 + akshare + baostock 库
